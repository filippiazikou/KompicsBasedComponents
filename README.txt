This project contains the assignemnts of ID2203 Distributed System, Advanced Course of KTH Royal Institute of Technology. 
It contains the following Components using Kompics framework:
1. Perfect Point to Point Link
2. Eventually Perfect Point to Point Link
3. Perfect Failure Detector
4. Eventually Perfect Failure Detector
5. Eventually Leader Detector
6. Lazy Probabilistic Broadcast
7. Basic Broadcast
9. Paxos Uniform Consensus
10. Read Impose Write Consult
11. Abortable Consensus
